\begin{sectionbox}{Enunciado}
Escriba una función que reciba como parámetros cuatro estudiantes, cada uno modelado como un diccionario con las llaves ``nombre'', ``código'' y ``nota''. 

La función debe retornar el mejor estudiante de acuerdo con la nota; en caso de empate, el estudiante con el nombre más largo; y en caso de persistir el empate, el estudiante con el código más reciente. Sin embargo, si ningún estudiante alcanza la nota mínima de 3.0, la función debe retornar un diccionario vacío.
\end{sectionbox}

\pagebreak

\begin{sectionbox}{Solución}
\begin{minted}[escapeinside=||]{python}
def elegir_mejor_entre_dos(est: dict, mejor: dict) -> dict:
    if (
        |\tikzmark{ortop}|est["nota"] > mejor["nota"]|\tikzmark{orright}|
        or (est["nota"] == mejor["nota"]
            and len(est["nombre"]) > len(mejor["nombre"]))
        or (est["nota"] == mejor["nota"]
            and len(est["nombre"]) == len(mejor["nombre"])
            and est["código"] > mejor["código"])|\tikzmark{orbot}|
    ):
        |\tms{replA}|mejor = est|\tme{replA}|
    return mejor

def elegir_mejor_estudiante(
    estudiante_1: dict,
    estudiante_2: dict,
    estudiante_3: dict,
    estudiante_4: dict,
) -> dict:
    mejor = estudiante_1
    |\tikzmark{seqtop}|mejor = elegir_mejor_entre_dos(estudiante_2, mejor)|\tikzmark{seqright}|
    mejor = elegir_mejor_entre_dos(estudiante_3, mejor)
    mejor = elegir_mejor_entre_dos(estudiante_4, mejor)|\tikzmark{seqbot}|

    if mejor["nota"] < 3:
        mejor = {}

    return mejor
\end{minted}
