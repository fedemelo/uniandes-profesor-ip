\gridquestion{13}{En una biblioteca digital, el código de cada uno de los libros debe satisfacer las siguientes reglas:
\begin{itemize}
  \item No debe contener letras minúsculas.
  \item Debe consistir de exactamente cinco caracteres.
  \item Debe iniciar con el caracter \texttt{C}.
\end{itemize}

Implemente una función que reciba cuatro códigos de libro como parámetros y retorne un diccionario con los códigos como llaves, cada uno con un booleano como valor que indique si el código es válido o no.

\textit{Consejo: No demore más de 17 minutos en este punto.}}
